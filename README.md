## Rodar a aplicação

Poder executar através do docker com o comando
 ```docker-compose up --build```

Ou simplesmente rodar pelo nest/cli
```npm run start```

Caso não tenha o nest cli instalado 
```npm i -g @nestjs/cli```

Documentação da API
``` http://localhost:3000/doc/ ```
