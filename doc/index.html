to use this api you must use the follow endpoint: http://projects.excenbit.com/api/



New investment

this resource allows you to create a new investment

Request headers

accept application/json

Body format

name string Required name of investment owner
invest_date string Required date of investment creation in format dd/mm/YYYY
amount double Required amount in format 0.000,00
POST
/investment/new/

Example of body

{
	"name": jonh wick,
	"invest_date": "02/02/2022",
	"amount": "5.000,00"
}

Response

status string result of transaction
name string name of owner
creation date string date of creation
aount double amount invested
investment id integer id of investment

Example response

HTTP/1.1 200 OK
Content-Type: application/json 
{
	"status": "ok",
	"name": jonh wick,
	"creation date": "02/02/2022",
	"amount": 1000.00,
	"investment id"	: 876
}

List of investment

this resource brings a list of all investments made

Request headers

accept application/json

Body format

name string Required name of investment owner
page integer Optional page number(limited to with 10 results per page)
POST
/investment/list/

Example of body

{
	"name": jonh wick,
	"page": "1"
}

Response

investment id integer id of investment
name string name of owner
amount string invested amount
date of creation string date of creation
status string status can be "active" or "withdrawn"
date of withdrawal string if it has not benn withdrawal is "null"
withdrawal amount string amount withdrawn

Example response

HTTP/1.1 200 OK
Content-Type: application/json 
[{
	"investment id": "10",
	"name": "josé mane",
	"amount": "450.00",
	"date of creation": "2020-02-17 00:00:00",
	"status": "active",
	"date of withdrawal": null,
	"withdrawn amount": "0.00"
},
...]

View investment

this resource shows the details of an investment

Query parameters

id id of investment
GET
/investment/view

Request example

http://endpoint/investment/view?id=45

Response

name of owner string name of owner
investment id string id of investment
status string status can be "active" or "withdrawn"
investiment date string date of creation
initial amount string amount invested
compound gain string tax that investment will pay every month
months invested string months since initial date
profit total string profit without tax
withdrawal tax string tax aplied to profit
tax total string total of tax over profit
free profit string profit final
amount total string total to withdrawal

Example response

HTTP/1.1 200 OK
Content-Type: application/json 
{
	"name of owner": "josé mane",
	"investment id": "20",
	"status": "active",
	"investment date": "17-12-2020",
	"withdrawn date": "",
	"initial amount": "450.00",
	"compound gain": "0.52%",
	"months invested": 14,
	"profit total": "33.89",
	"withdrawal tax": "18.5%",
	"tax total": "6.27",
	"free profit": "27.62",
	"amount total": "511.51"
}
